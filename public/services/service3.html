影像辨識

人臉辨識除了可計算人流，未來還可輸入並記憶資料提供特定名單辨別，提高資安防護

Real face detection system

A conventional face identification system can be manipulated by using a photo for the identification. We study and develop "real face detection" to be able to tell between a photo and a real face, and is able to prevent any security issues caused by using a video or photograph.
